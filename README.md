### Keylogger Python ###

Educational keylogger project written in Python.
Projet de keylogger en Python à des fins pédagogiques.

Created for a Cybersecurity course to understand keyboard event handling.
Créé pour un cours de Cybersécurité afin de comprendre la capture des frappes clavier.

Learning Python, Linux, HTML/CSS, and Cybersecurity fundamentals through hands-on projects
